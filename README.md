### Home Page (`home.html`)

The landing page of the portfolio website. It introduces visitors to who I am and highlights my role as a Python Full Stack Developer. This page includes:

* Hero section with introduction
* Profile image
* Call-to-action buttons
* Quick navigation to other sections

### About Page (`about.html`)

Provides detailed information about me, my background, and my journey in web development.

* Personal introduction
* Educational background
* Career goals
* Professional summary

### Skills Page (`skills.html`)

Displays the technologies and tools I use for development.

* Python
* Django
* HTML5
* CSS3
* JavaScript
* Git & GitHub
* Database technologies

### Projects Page (`projects.html`)

Showcases my completed projects with descriptions and technologies used.

* Project title
* Project overview
* Technologies used
* GitHub repository links
* Live demo links (if available)

### Contact Page (`contact.html`)

Allows visitors to connect with me.

* Contact form
* Email information
* Social media links
* Professional networking links

### Base Template (`base.html`)

The main template used throughout the website.

* Common navigation bar
* Footer section
* Shared styling and layout
* Template inheritance using Django's `{% extends %}`

This structure helps maintain a clean and reusable codebase while ensuring a consistent user experience across all pages.
